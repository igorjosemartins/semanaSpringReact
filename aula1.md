## Aplicação

Front-end <---- requisições web (HTTP/JSON) ----> Back-end + Banco de Dados (server)



### Front-end

- Linguagens
   -> HTML/CSS + JavaScript/TypeScript

- Framework
   -> ReactJS



### Back-end

- Linguagens
   -> Java + SQL

- Framework
   -> Spring Boot

- DB
   -> H2

   -> Usamos o framework Spring Initializr para nos ajudar a criar o projeto em Spring
